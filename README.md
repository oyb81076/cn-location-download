# 省市区统计局爬虫
安装 ``yarn install``
# 帮助
`` yarn build --help``
# usage
``yarn build --depth=2 ``
# options
```
Options:
  -d, --depth [depth]  搜索深度1:省,2:市,3:区 (default: "1")
  -u, --url [url]      网址 (default: "http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2017/index.html")
  -h, --help           output usage information
```
